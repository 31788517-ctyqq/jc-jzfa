# ⚽ 功守道 · 八阵图 V24.0 开发结构文档

## 1. 项目目标

构建足球赛事量化分析系统，通过攻、防、道三维模型完成：

-   实力分析
-   大小球分析
-   净胜球分析
-   比分矩阵预测
-   让球方向裁决

------------------------------------------------------------------------

# 2. 系统架构

``` text
数据输入层
 ├── JSON赛事数据
 ├── 历史比赛数据
 └── 赔率数据

字段映射层
 ├── 原子字段解析
 ├── 正则提取
 └── 标准变量映射

量化计算层
 ├── 第一维：实力模型
 ├── 第二维：大小球模型
 ├── 第三维：净胜球模型
 └── 第四维：比分矩阵模型

输出层
 ├── Markdown报告
 ├── API返回
 └── 前端展示卡片
```

------------------------------------------------------------------------

# 3. 数据字段定义

## 输入JSON标准

``` json
{
 "homeWinGap_2":"",
 "homeWinGap_1":"",
 "homeLoseGap_1":"",
 "homeLoseGap_2":"",
 "awayWinGap_2":"",
 "awayWinGap_1":"",
 "awayLoseGap_1":"",
 "awayLoseGap_2":""
}
```

## 核心变量映射

  变量       说明
  ---------- ------------
  Adv_攻     进攻优势
  Adv_守     防守优势
  Total_战   综合战力
  Anchor     动态锚点
  λ_total    总进球期望
  Diff_exp   净胜球预期

# 4. 第一维：实力分析

## 子模型

### 4.1 进攻优势

计算：

-   赢球格局对冲
-   攻击效率对冲
-   进球厚度分布

输出：

``` ts
{
 attackAdvantage:"+20%"
}
```

------------------------------------------------------------------------

### 4.2 防守优势

计算：

-   输球容错
-   防御效率
-   零封能力

输出：

``` ts
{
 defendAdvantage:"-10%"
}
```

------------------------------------------------------------------------

### 4.3 综合攻守优势

输出：

``` ts
{
 totalAdvantage:"+50%"
}
```

------------------------------------------------------------------------

# 5. 第二维：大小球模型

输入：

-   主客权重
-   场地烈度
-   历史大球率
-   最近交锋基因

计算：

1.  权重放大器
2.  总进球期望
3.  区间锁定

输出：

``` ts
{
 totalGoalExpected:2.85,
 range:"2-3球"
}
```

------------------------------------------------------------------------

# 6. 第三维：净胜球模型

流程：

``` text
预期进球
 ↓
净胜球差
 ↓
动态锚点
 ↓
7场阈值过滤
 ↓
最终净胜球判断
```

输出：

``` ts
{
 diff:"+1.2",
 result:"主队强势"
}
```

# 7. 第四维：比分矩阵模型

三重物理锁：

## 锁一

## 锁二

## 锁三

净胜球限制

## 泊松矩阵

``` text
P(score)=Poisson(Home)
×
Poisson(Away)
×
历史修正
×
实力修正
```

输出：

``` json
[
 {"score":"1:0","rate":"18.2%"},
 {"score":"2:1","rate":"15.3%"}
]
```
